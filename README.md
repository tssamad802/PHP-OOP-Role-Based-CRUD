# PHP OOP Role-Based CRUD

This project is a **PHP OOP-based web application** demonstrating **role-based access control** and **CRUD operations** for posts and categories.  

## Features

- **Role-Based Access:**  
  - Admin: Full access to all features.  
  - Editor: Limited access based on assigned roles.  

- **Posts & Categories CRUD:**  
  - Create, Read, Update, Delete operations for posts and categories.  
  - Categories have **slugs**, and posts are dynamically linked to category slugs.  

- **Dynamic Linking:**  
  - Every post is associated with a category slug.  
  - Categories show linked posts via slugs.  

- **OOP Approach:**  
  - Organized, reusable PHP classes.  
  - Clean structure for scalability and maintenance.  

## Tech Stack

- PHP (OOP)  
- MySQL  
- HTML/CSS (optional: JS for enhancements)
